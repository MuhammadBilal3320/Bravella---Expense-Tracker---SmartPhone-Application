## About Bravella

Bravella is an offline-first expense tracking mobile application built using React Native and Expo.

The app is designed to help users manage their finances by tracking income, expenses, and borrowed money with a strong focus on monthly organization and financial clarity. All data is stored locally on the device, ensuring complete privacy and uninterrupted usage without any internet connection.

## Features

• Track income, expense, and borrow transactions  
• Handle partial and full borrow repayments  
• Automatic month-based transaction management  
• Close month functionality to lock completed months  
• View and manage closed month history  
• Export current month or closed months as PDF reports  
• Clear indicators for paid and unpaid transactions  
• Clean and distraction-free user interface

## Installation & Usage

Prerequisites:
• Node.js (v18 or higher)  
• Expo CLI  
• Android device or emulator  

Steps:
1. Clone the repository  
   git clone https://github.com/your-username/bravella.git ](https://github.com/MuhammadBilal3320/Bravella---Expense-Tracker---SmartPhone-Application

2. Navigate to the project directory  
   cd bravella  

3. Install dependencies  
   npm install  

4. Start the application  
   npx expo start  

Scan the QR code using Expo Go or run on an Android emulator.


## Build

Android APK:
npx expo run:android  

Android App Bundle:
npx expo prebuild
